How Validation Works

The validation pipeline operates in 4 sequential stages, each filtering designs before passing them forward.

Stage 1 — Core Validator
Assigns a composite viability score. Designs scoring above 0.70 proceed. Below this threshold, the design is immediately rejected.
Stage 2 — Cross-Domain Validator
Applies domain-specific checks in order: Biology → Materials → Physics → Chemistry. Failure in any domain halts processing immediately.
Stage 3 — Accuracy Validator
Compares surrogate model predictions against full-physics ground truth. Requires accuracy >95%, FP rate <5%, FN rate <10%.
Stage 4 — Failure Analyzer
For rejected designs, diagnoses the root cause: Physics Issue, Boundary Issue, Data Issue, Domain Mismatch, or Threshold Breach. Returns actionable suggestions.
